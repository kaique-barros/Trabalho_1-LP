Métodos de cada classe:

Cartão de crédito:
    -Retornar o numero (particionado e completo apenas com a senha) 
    -Validar a compra
    -Validar as informações do dono

Cliente:
    -Retornar nome, email e telefone
    -Retornar cpf, rg e formas de pagamento apenas com senha
    -Adicionar/remover formas de pagamento ✅
    -Adicionar/remover endereço ✅

Compra: 
    -Validar as info do cliente, do cartão e do produto
    -Obter a data
    -Gerar comprovante

Produto 
    -Retorna todas informações para o Vendedor
    -Retorna apenas nome, descrição e imagem para cliente
    -Atualiza o estoque
    -Realiza pedido de novo estoque

Repor Estoque
    - Envia email para o fornecedor pedindo mais unidades
    - Atualiza o estoque quando for confirmado que o pedido chegou 
    - Emite comprovante

Fornecedor
    - Disponível apenas para o vendedor, retorna todos os dados

Vendedor
    - Cadastrar Cliente ✅
